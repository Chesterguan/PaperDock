A Zotero companion · macOS

Ask your own library.

Question the papers you already keep in Zotero — and get answers with citations you can click to open the source PDF. Fact-check a claim, or vet a reference for fabrication. Runs on your Mac; your library never leaves it.

Three ways to work
ASK

Grounded answers

Type a question; get an answer built only from your papers, with inline citations that open the PDF in Zotero. Follow-ups stay in one thread.

CHECK CITATION

Does it hold up?

Paste a claim; PaperDock judges whether your papers actually support it — with the evidence quoted and cited. Target one paper or the whole set.

SUPPORTED PARTIAL NOT SUPPORTED INSUFFICIENT
VERIFY REFERENCE

Real, or made up?

Paste a reference; PaperDock checks it against CrossRef and flags the likely-fabricated citations that LLM-written text loves to invent.

Built the quiet way

Local-first. Your papers stay put.

Personal libraries embed on your own machine. Only a shared Zotero group library uses a shared vector store — and only if you set one up.

Bring your own LLM.

OpenAI, a local Ollama, or a gateway like UF NaviGator — anything OpenAI-compatible. No keys are baked into the app.

Open source. MIT.

A thin native shell (Tauri + Leptos) around PaperQA. Read it, fork it, send a PR.

Get started
  1. Download the DMG and drag PaperDock into Applications.
  2. First open: right-click → Open (it's unsigned — only needed once).
  3. One-time setup runs on first launch (~1–2 min, needs internet).
  4. In Zotero, turn on Settings → Advanced → “Allow other applications on this computer to communicate with Zotero.”
  5. Pick a collection, add your LLM key in ⚙ Settings, and ask.

Requirements

  • macOS — Intel or Apple Silicon
  • Zotero 7+, running, local connection on
  • An LLM key (OpenAI / NaviGator / …) or local Ollama
  • Internet for the one-time setup
